# 📦 Archived: Daily Dashboard (Watchlist App)

This directory contains the original **mphinance Daily Dashboard** — a Vite + React + TypeScript app with:

- Real-time watchlist management
- Trading journal
- Performance analytics / charts
- Polygon.io API integration

## Why it's here

The `momentum-hub` repo was repurposed as a **unified task tracker** for all 20+ workstreams.
The dashboard code was preserved here in case any watchlist features or components are needed later.

## Can it still run?

It *should* work if you install and run from this directory:

```bash
cd archive/dashboard
npm install
npm run dev
```

> **Note:** Requires a Polygon.io API key in the environment. See `README.md` in this folder for full setup.
